# marvin-cowsay

The first plugin for [Marvin](https://github.com/marvin-js/marvin)

# Contents

- [Install](#install)
- [Usage](#usage)

## Install

With yarn:

```
yarn global add marvin-cowsay
````

With npm:

```
npm install marvin-cowsay --global
```

## Usage

```
// .marvin on project folder

cowsay hello world
```
